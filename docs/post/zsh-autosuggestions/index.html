How to Make ZSH Auto-Suggestions

  • 1/12/2022
  • One-minute read

Background

ZSH is a very popular Unix shell built on top of bash, and the Bash shell will ultimately be dropped from the macos core. I chose to use oh-my-zsh and customize the color style. Here is a record of the plugin usage using auto-suggestions as an example.

Process

Install zsh-autosuggestions very easy, you can view here, below are my steps:

  1. Download the plugin to oh-my-zsh default plugins folder:
1

git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
  1. Add the plugins:
1
2
3
4

# in ~/.zshrc
plugins=( 
    zsh-autosuggestions
)

Don’t forget excute

1

source ~/.zshrc
  1. Now you can use it! But if you want to use “TAB” key to choose, need to add keybind in .zshrc file:
1

bindkey '\t' end-of-line

Remember “source” the file let it available. Preview: My plugin list:

1
2
3
4
5
6
7
8
9

plugins=(
    colored-man-pages
    zsh-autosuggestions
    colorize
    git
    sudo
    vi-mode
    z
)

Summarize

Command-line tools are loved by many developers because of their powerful functions, and the products are becoming more and more abundant, such as the recently popular Fig. However, using oh-my-zsh can make some personalized adjustments to suit my preferences 😄.
